# 課題1
## 回答
![ER図](./sushi1.png)

- 注文詳細テーブルで商品テーブルとは別で消費税、税抜き単価、税込み単価を用意する。
  - 商品の価格変更でレコードの価格フィールドが更新されると、過去注文情報の金額と差異が生じるため
- セットメニューとお好み寿司との区別は商品種別のマスタテーブルを作成する。
